# Project Name: Keen Keeper

## 📖 Description

KeenKeeper is a simple and user-friendly web application where users maintain and track their friendships. Users can monitor interactions, and stay connected through call, text, or video check-ins.

## 🛠️ Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- Recharts
- React Toastify
- Vite

## ✨ Key Features

1. **Track Interactions** – Users can log Call, Text, and Video interactions.
2. **Timeline View** – Users can view all activities in a timeline format.
3. **Analytics Dashboard** – Users can see an interaction summary using a pie chart.
